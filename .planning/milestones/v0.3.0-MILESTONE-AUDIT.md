The Scipy integration demonstrates robust end-to-end flows:
1.  **Discovery:** AI agents can discover hundreds of SciPy functions with rich parameter schemas.
2.  **Execution:** Functions are executed in isolated environments with proper dtype promotion.
3.  **Accuracy:** Outputs for common operations (Gaussian, T-test) match native Scipy perfectly.
4.  **Data:** N-D artifacts flow between SciPy and other tool packs (Trackpy/Cellpose).
