<task type="auto">
  <name>Task 2: Implement StarDistAdapter and runtime meta.list/meta.describe wiring</name>
  <files>
    tools/stardist/bioimage_mcp_stardist/dynamic_discovery.py
    tools/stardist/bioimage_mcp_stardist/entrypoint.py

  <action>
- Create `tools/stardist/bioimage_mcp_stardist/dynamic_discovery.py` implementing a `StarDistAdapter` with a `discover(source_config)` method compatible with `bioimage_mcp.registry.dynamic.discovery.discover_functions`.
- Discovery scope (keep tight; only what we will execute later):
  - `stardist.models.StarDist2D.from_pretrained`
  - `stardist.models.StarDist3D.from_pretrained`
  - `stardist.models.StarDist2D.predict_instances`
  - `stardist.models.StarDist3D.predict_instances`
  - Optionally include `train` methods ONLY if you will implement them in Plan 16-03.
- For each discovered callable, produce `FunctionMetadata` with:
  - `fn_id` that matches the underlying callable (use a stable prefix like `stardist.models.StarDist2D.predict_instances`).
  - `io_pattern` set to the closest available pattern:
    - constructors: `pure_constructor`
    - predict_instances: `image_to_labels_and_json`
- In `entrypoint.py`, implement `handle_meta_list` by:
  - Loading and validating the manifest via `ToolManifest` (same as trackpy does).
  - Calling `discover_functions(manifest, {"stardist": StarDistAdapter()}, cache=IntrospectionCache(...), project_root=...)`.
  - Returning the minimal meta.list entry shape required by `parse_meta_list_result` (id/name/summary/module/io_pattern) and top-level `tool_version` and `introspection_source`.
- Implement `handle_meta_describe` for `target_fn` by:
  - Resolving the callable (module import + getattr) for supported targets.
  - Using `bioimage_mcp.registry.dynamic.introspection.Introspector().introspect(...)` to obtain a `FunctionMetadata` with `parameters`.
  - Converting `parameters` to JSON Schema via `bioimage_mcp.registry.engine.DiscoveryEngine.parameters_to_json_schema(...)`.
  - Returning `{ok: True, result: {params_schema, tool_version, introspection_source}}` where:
    - `tool_version` comes from `importlib.metadata.version("stardist")` when available.
    - `introspection_source` is a stable string like `python_api`.
  - If `target_fn` is unknown, return `{ok: False, error: "target_fn not supported: ..."}`.
  </action>
  <verify>python -m compileall tools/stardist/bioimage_mcp_stardist/dynamic_discovery.py</verify>
