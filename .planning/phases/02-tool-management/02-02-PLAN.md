<task type="auto">
  <name>Task 1: Refactor install.py for extensibility</name>
  <files>src/bioimage_mcp/bootstrap/install.py</files>
  <action>
Refactor install() to dynamically discover and install tools:

1. **Tool discovery function:**
   ```python
   def discover_available_tools() -> dict[str, Path]:
       """Return {tool_name: env_yaml_path} for all tools in envs/ directory."""
   ```
   - Scan `envs/` directory for `bioimage-mcp-*.yaml` files
   - Parse tool name from filename (e.g., `bioimage-mcp-cellpose.yaml` -> `cellpose`)
   - Return mapping of tool_name -> yaml_path

2. **Profile definitions:**
   ```python
   PROFILES = {
       "cpu": ["base", "cellpose"],  # Default CPU profile
       "gpu": ["base", "cellpose"],  # GPU adds CUDA post-install
       "minimal": ["base"],          # Just the base
   }
   ```

3. **Refactor install() signature:**
   ```python
   def install(
       *,
       tools: list[str] | None = None,
       profile: str | None = None,
       force: bool = False,
   ) -> int:
   ```

4. **Install logic:**
   - If profile given: expand to tool list from PROFILES
   - If tools given: use those (validate they exist)
   - If neither: default to profile="cpu"
   - Always prepend "base" if not in list (base is required)
   - For each tool:
     a. Check if env already exists (skip if exists and not force)
     b. Run conda/micromamba env create/update
     c. Track success/failure
   - Print summary at end: "Installed: X, Skipped: Y, Failed: Z"

5. **GPU handling:** Keep existing GPU logic for cellpose (remove cpuonly, add pytorch-cuda)

6. **Output format:**
   - Print each tool being installed: "Installing cellpose..."
   - On skip: "cellpose already installed (use --force to reinstall)"
   - On failure: "Failed to install cellpose: {error}"
   - Summary line at end

Do NOT add spinners or colors yet - keep it simple text output.

  </verify>

Update the install subparser in cli.py:

1. Change from positional-only profile to flexible args:
   ```python
   install = subparsers.add_parser("install", help="Install tool environments")
   install.add_argument("tools", nargs="*", help="Specific tools to install (e.g., cellpose tttrlib)")
   install.add_argument("--profile", choices=["cpu", "gpu", "minimal"], help="Install a predefined profile")
   install.add_argument("--force", action="store_true", help="Reinstall even if already exists")
   ```

2. Update _handle_install to pass new args:
   ```python
   def _handle_install(args: argparse.Namespace) -> int:
       from bioimage_mcp.bootstrap.install import install
       return install(
           tools=args.tools if args.tools else None,
           profile=args.profile,
           force=args.force,
       )
   ```

3. Add validation: error if both tools AND profile specified (mutually exclusive)

Usage examples after change:
- `bioimage-mcp install` (default: --profile cpu)
- `bioimage-mcp install --profile gpu`
- `bioimage-mcp install cellpose`
