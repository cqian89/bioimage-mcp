- `inspect_module(module: str, search_paths: list[pathlib.Path]) -> StaticModuleReport`:
  - Uses `griffe` to load module/package WITHOUT importing tool code.
  - Extracts fully qualified module path(s) and callable definitions.
  - Exposes enough info for later steps: callable name, qualified name, docstring text, signature params (names + defaults as strings), and the callable source text (or a best-effort fallback).

  - Returns sha256 hex digest; stable across runs.

- `normalize_json_schema(schema: dict) -> dict`:
  - Produces deterministically ordered schema output (sort object keys; ensure `required` is a stable list ordering).
  - Does not change semantics; only canonicalizes.

Hard requirements:
- No importing tool-pack modules in the core server process.
- Full module paths are preserved (no truncation to last segment).
- New code follows repo conventions (`from __future__ import annotations`, typing, ruff).
