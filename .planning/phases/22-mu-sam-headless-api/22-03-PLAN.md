  - Accept BioImageRef/LabelImageRef inputs as dict refs and load to numpy using the same strategy as SkimageAdapter (bioio first; preserve native dims).

  - Convert JSON-friendly lists (points/boxes) into numpy arrays when calling upstream micro_sam.
  - If mask prompting/refinement is supported, allow an optional mask prompt via LabelImageRef input.

  - If images are too large, fail with an actionable error (do not auto-downsample by default).
- Axis policy:
  - Preserve axes metadata from the input artifact when possible.
