  <what-built>
Complete artifact_info preview system:
- BioimageRef: base64 PNG preview with projection and channel selection
- LabelImageRef: colormap rendering with region count and centroids
- TableRef: markdown preview with dtypes
- ObjectRef: native_type and repr preview
  </what-built>
  <how-to-verify>
1. Start server: `bioimage-mcp serve`

2. Test image preview via MCP client or direct API:
   ```python
   # Import a test image
   result = run("base.io.import_image", {"path": "datasets/mitosis.ome.tif"})
   ref_id = result["outputs"]["image"]["ref_id"]
   
   # Get preview
   info = artifact_info(ref_id, include_image_preview=True)
   
   # Verify base64 decodes to viewable image
   import base64
   from PIL import Image
   from io import BytesIO
   img_data = base64.b64decode(info["image_preview"]["base64"])
   img = Image.open(BytesIO(img_data))
   img.show()  # Should display the preview
   ```

3. Test table preview:
   ```python
   # Import a CSV
   result = run("base.io.import_table", {"path": "datasets/smoke_tmp/stats.csv"})
   ref_id = result["outputs"]["table"]["ref_id"]
   
   info = artifact_info(ref_id, preview_rows=5)
   print(info["table_preview"])  # Should show markdown table
   print(info["dtypes"])  # Should show column types
   ```

4. Verify all preview parameters work as documented in CONTEXT.md

5. Check that previews are omitted (not errored) when generation fails
