<task type="auto">
  <name>Task 1: Create preview utility module</name>
  <files>src/bioimage_mcp/artifacts/preview.py, pyproject.toml</files>
  <action>
Create `src/bioimage_mcp/artifacts/preview.py` with the following utilities:

1. `apply_projection(array: np.ndarray, axis: int, method: str) -> np.ndarray`
   - Methods: "max", "mean", "sum", "min", "slice"
   - For "slice": take middle index by default
   - Returns reduced array

2. `reduce_to_2d(array: np.ndarray, dims: list[str], projection: str | dict, slice_indices: dict | None, channel: int | None) -> np.ndarray`
   - Reduce N-D array to 2D for preview
   - Default projection order: reduce Z first, then T
   - Handle channel selection via `channel` parameter
   - Skip already-2D arrays

3. `normalize_to_uint8(array: np.ndarray) -> np.ndarray`
   - Map array values to 0-255 range
   - Handle edge case where min == max

4. `resize_preserve_aspect(array: np.ndarray, max_size: int) -> np.ndarray`
   - Resize so largest dimension is max_size
   - Preserve aspect ratio
   - Use nearest-neighbor or bilinear interpolation

5. `encode_png_base64(array: np.ndarray) -> str`
   - Convert 2D uint8 array to PNG
   - Return base64-encoded string
   - Use Pillow for encoding

6. `generate_image_preview(path: Path, dims: list[str], shape: list[int], dtype: str, *, max_size: int = 256, projection: str | dict = "max", slice_indices: dict | None = None, channel: int | None = None) -> dict | None`
   - Main entry point for BioImageRef preview
   - Load image data using bioio (lazy slicing where possible)
   - Apply reduce_to_2d, normalize_to_uint8, resize_preserve_aspect, encode_png_base64
   - Return `{"base64": ..., "format": "png", "width": ..., "height": ...}`
   - Return None on any failure (fail silently per CONTEXT.md)

Use imports: `from __future__ import annotations`, numpy, Pillow, bioio.
Handle errors gracefully - catch exceptions and return None rather than propagating.

Also add `pillow` to `[project].dependencies` in `pyproject.toml` (required for PNG encoding).
  </action>
