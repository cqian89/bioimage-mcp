# Hazelnut FLIM PTU Dataset

## Description
A Time-Tagged Time-Resolved (TTTR) FLIM dataset of a hazelnut sample in PicoQuant PTU format. This file contains photon arrival time data suitable for FLIM phasor analysis workflows.
