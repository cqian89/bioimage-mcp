# Trackpy Example Data

This directory contains minimal sample data vendored from the [trackpy-examples](https://github.com/soft-matter/trackpy-examples) repository for use in smoke tests.

## Provenance

- **Source:** [soft-matter/trackpy-examples](https://github.com/soft-matter/trackpy-examples)
- **Repo URL:** `https://github.com/soft-matter/trackpy-examples/tree/master/sample_data/bulk_water`
- **Reference Doc:** [Trackpy Walkthrough](https://soft-matter.github.io/trackpy/v0.7.0/tutorial/walkthrough.html)

## Files

- `bulk_water/frame000.png`: Original frame 0 from the `bulk_water` dataset in `trackpy-examples` (actually named `bulk_water_000.png` upstream).
- `bulk_water/frame000_green.ome.tiff`: Derived grayscale version of `frame000.png`, extracting the green channel (index 1) as used in the Trackpy walkthrough tutorial.
