# Seminal Receptacle FLIM SDT Dataset

## Description
A Becker & Hickl SDT file containing Time-Correlated Single Photon Counting (TCSPC) data from a seminal receptacle sample. This is a single FLIM image suitable for testing SDT file loading and phasor analysis workflows.
